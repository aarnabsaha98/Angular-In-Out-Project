Directives

Directives are use to change the behavior and appearance of DOM Element


A. Directives can implement all lifecycle hooks.
B. Directives can not have template
There are two Types ::
1. Structural Directives : that has ability to modify i.e. add or remove element from the DOM element are structural marked as (* at beginning of declaration i.e. *ngFor, *ngIf)
2. Attribute Directives :: add extra logic or modify the DOM But not able to add or remove. (ngClass, ngStyle)
